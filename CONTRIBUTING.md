Project Workflow (SDLC + CI)

The Lead is the only one that can push directly to the main branch.  Everyone else must work through branches and Pull Requests.

For the Designer, SWE, and Tester
1. Fork the repo
2. Create a branch for your task
3. Commit your work to your branch
4. Push your branch
5. Open a Pull Request
5. Wait for Lead to review and merge

For the Lead
-Review Pull Requests
-Request fixes if needed
-Merge approved changes into main
